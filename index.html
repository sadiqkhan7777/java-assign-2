WHAT IS IF AND ELSE CONDITION

if aur else javascript key ak condition hai jo kay program ko descions making kar nay key ijazat day ta hay agar conditon true hogi toh if block ka code run hota hay or agar condtion false hoto else ka block code run hota hay

Example:-

let age = 18; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }
